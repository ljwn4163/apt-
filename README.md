# 🏠 SAIT: Seoul Apartment Index Token

> **주택가격 ‘지수’를 상장해 롱·숏이 가능한 현금결제 파생시장을 만든다.**  
> 가격 안정화(공매도 유입) · 리스크 헷지(건설·PF·임대) · 투명성(온체인 공개)

---

## 🧭 프로젝트 개요

SAIT 프로젝트는 서울 및 주요 지역의 **주택가격 지수**를 기초자산으로 한  
**현금결제형 파생시장**을 구축하기 위한 프로토타입입니다.

**핵심 목표**
- 🧩 **가격 안정화:** 공매도·롱숏 메커니즘을 제도권 안으로 유입  
- 🛡️ **리스크 헷지:** PF대출·건설사·임대사업자의 가격변동 위험 완화  
- 🔍 **투명성:** 오라클·정산·보험기금 내역을 온체인 공개

## 📊 지수 및 오라클 설계

**데이터 소스**
- 1차: KB, 한국부동산원, 국토부 등 복수 공식지수  
- 2차(nowcast): 실거래가, 호가, 전세가, 거래량, 헤도닉모델  
- 업데이트 주기: 공식지수 월간 / Nowcast 일·시간 단위  

**오라클 파이프라인**
다중소스 → 미디안 집계 → Winsorize → TWAP → Commit-Reveal


- 공급자 가중치 상한: ≤ 25%  
- 모든 로그 및 해시 커밋 기록을 온체인에 공개  

**Fixing / 정산**
- 공식지수 발표 ±72h TWAP 가중평균  
- “Exchange Monthly Fix” 단일값으로 정산

## 💰 마진·청산 파라미터

| 티어 | 명목 한도 | 최대 레버리지 | 초기증거금 | 유지증거금 |
|------|-------------|----------------|--------------|--------------|
| T1 | ≤ 1억원 | 10x | 10% | 0.5% |
| T2 | ≤ 5억원 | 7x | 14.3% | 0.7% |
| T3 | ≤ 20억원 | 5x | 20% | 1.0% |

**청산 예시**
> 진입 100p, IM=10%, MMR=0.5% → 청산가 ≈ 90.45p (하락 약 −9.5%)

**가격 안정장치**
- 동적 프라이스밴드 (±5×ATR)
- 서킷브레이커 (급변시 30초 정지)
